# 小杜庄村官方网站

这是河南省西华县逍遥镇小杜庄村的官方宣传网站，展示村庄风貌、特色产业、杰出人才、龙头企业等信息。

## 访问地址

网站已发布到 GitHub Pages：[https://91coin.github.io/xiaoduzhuang](https://91coin.github.io/xiaoduzhuang)

## 网站特色

✅ 现代化设计，高端大气，符合美丽乡村形象
✅ 完整的功能模块：
- 首页形象展示
- 村庄概况介绍
- 特色农业（甜瓜、辣椒、西瓜）展示
- 杰出人才介绍（杜国楹等优秀乡贤）
- 龙头企业（晟源农业合作社）宣传
- 美丽乡村相册
- 联系信息和地图定位

✅ 响应式设计，完美适配电脑、平板、手机等各种设备
✅ 丰富的交互效果：
- 平滑滚动
- 图片悬停放大
- 卡片悬停动画
- 回到顶部按钮
- 导航栏滚动效果

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome 图标
- 原生 JavaScript
- 纯静态页面，无需服务器支持

## 本地运行

直接在浏览器中打开 `index.html` 文件即可查看。
